Cara deploy ke Netlify:
1. Ekstrak ZIP ke folder lokal.
2. Buka https://app.netlify.com, login atau daftar.
3. Pilih 'Sites' -> 'New site from Git' (atau drag & drop folder ke 'Sites' area untuk deploy manual).
4. Jika drag & drop, pilih folder ini (isi index.html di root).
5. Setelah deploy selesai, buka URL yang diberikan Netlify.

Catatan:
- Ganti images/favicon.png dengan file favicon sesungguhnya bila perlu.
- Untuk mengganti video YouTube, edit index.html pada bagian iframe src.
- Untuk mengupload video .mp4, tempatkan file ke folder /assets/videos dan ubah data-type ke 'mp4' di galeri video.
